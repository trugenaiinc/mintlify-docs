---
title: "n8n Workflows"
---

TruGen supports no-code workflow automation through an official n8n integration. You can use it to create agents, process events, and connect your conversational pipelines to the rest of your stack.

## Get Started

<Steps>
  <Step title="Install and configure the n8n integration">
    Install the official TruGen n8n community node from our repository:
    [trugenai/n8n-nodes-trugen](https://github.com/trugenai/n8n-nodes-trugen), then add your API credentials in n8n.
  </Step>
  <Step title="Review node capabilities and setup details">
    Review the README in the official node repository for setup steps, supported operations, and example workflow patterns.
  </Step>
  <Step title="Build from templates">
    Use these ready-made TruGen n8n workflow examples:
    - [HR Interview Agent](https://github.com/Bharath8080/Workflows_Trugen-AI/tree/main/hr_interview_agent) - Automate candidate screening and post-call analysis.
    - [Sales Qualification Agent](https://github.com/Bharath8080/Workflows_Trugen-AI/tree/main/sales_qualification_agent) - Run BANT-style lead qualification and CRM sync.
    - [Sentiment Analysis Agent](https://github.com/Bharath8080/Workflows_Trugen-AI/tree/main/sentiment_analysis_agent) - Capture CSAT and sentiment insights from conversations.
  </Step>
  <Step title="Connect workflows to your production flow">
    Extend your automation by pairing n8n workflows with managed agents and voice-to-video experiences.
  </Step>
</Steps>
