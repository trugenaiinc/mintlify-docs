title: "Livekit Integration"
---

Give a face to your Livekit agents with our TruGen Avatars, using our official Livekit Agents plugin.

## Get started

<Tabs>
  <Tab title="Python">
    <Steps>
      <Step title="Install the plugin">
        Install the TruGen plugin from Livekit registry, using the following command.
```bash
        uv add "livekit-agents[trugen]~=1.3" # When using UV
```
```bash
        pip install "livekit-agents[trugen]~=1.3" # When using PIP
```
      </Step>
      <Step title="Generate an API key">
        1. Using TruGen.AI [Developer Platform](https://app.trugen.ai), generate an API key from our developers page.
        2. Set the newly generated API key as `TRUGEN_API_KEY` environment variable in your `.env` file.
      </Step>
      <Step title="Adding Avatar session to Livekit Agent">
        1. To your existing Livekit agent, import the `trugen` plugin from `livekit.plugins`.
```python
        from livekit.plugins import trugen
```
        2. Instantiate a new instance of `AvatarSession` from the `trugen` plugin and set the avatar id.
```python
        avatar_id = os.getenv("TRUGEN_AVATAR_ID") or "45e3f732"
        trugen_avatar = trugen.AvatarSession(avatar_id=avatar_id)
```
        3. Start the `trugen_avatar` session using the start method, while passing the current agent session and room context.
```python
        await trugen_avatar.start(session, room=ctx.room)
```
        #### Complete Example
        <CodeGroup>
```python Python highlight={1-2,8-10}
        from livekit.plugins import trugen

        async def entrypoint(ctx: JobContext):
            session = AgentSession(
              # Your existing configuration
            )

            avatar_id = os.getenv("TRUGEN_AVATAR_ID") or "45e3f732"
            trugen_avatar = trugen.AvatarSession(avatar_id=avatar_id)
            await trugen_avatar.start(session, room=ctx.room)

            await session.start(
              # Start agent session as usual
            )

````
        </CodeGroup>
        4. Gracefully shutdown Livekit agent; use `ctx.room.disconnect()` to close the room connection or use `ctx.shutdown()` to end the `JobContext`.
        <Tip>This step ensures that your TruGen Avatar session shuts down correctly. We automatically stop the session when the room is disconnected or when the primary participant is disconnected.</Tip>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Node.js">
    <Steps>
      <Step title="Install the plugin">
        Install the TruGen plugin from the npm registry using the following command.
```bash
        pnpm add @livekit/agents-plugin-trugen
````

      </Step>
      <Step title="Generate an API key">
        1. Using TruGen.AI [Developer Platform](https://app.trugen.ai), generate an API key from our developers page.
        2. Set the following environment variables in your `.env` file.

```env
        TRUGEN_API_KEY=your_api_key
        TRUGEN_AVATAR_ID=your_avatar_id
        LIVEKIT_URL=your_livekit_url
        LIVEKIT_API_KEY=your_livekit_key
        LIVEKIT_API_SECRET=your_livekit_secret
```

      </Step>
      <Step title="Adding Avatar session to Livekit Agent">
        1. Import and instantiate `AvatarSession` with your avatar credentials.

```javascript
const avatarSession = new AvatarSession({
  avatarId: process.env.TRUGEN_AVATAR_ID,
  apiKey: process.env.TRUGEN_API_KEY,
});
```

        2. Start the session inside your agent flow, passing your LiveKit credentials.

```javascript
await avatarSession.start(agentSession, room, {
  livekitUrl: process.env.LIVEKIT_URL,
  livekitApiKey: process.env.LIVEKIT_API_KEY,
  livekitApiSecret: process.env.LIVEKIT_API_SECRET,
});
```

        #### Complete Example
        <CodeGroup>

```javascript Node.js highlight={1,6-13}
const { AvatarSession } = require("@livekit/agents-plugin-trugen");

async function entrypoint(agentSession, room) {
  // Your existing agent session configuration above

  const avatarSession = new AvatarSession({
    avatarId: process.env.TRUGEN_AVATAR_ID,
    apiKey: process.env.TRUGEN_API_KEY,
  });

  await avatarSession.start(agentSession, room, {
    livekitUrl: process.env.LIVEKIT_URL,
    livekitApiKey: process.env.LIVEKIT_API_KEY,
    livekitApiSecret: process.env.LIVEKIT_API_SECRET,
  });

  // Continue with your agent session start as usual
}
```

        </CodeGroup>
        3. Once started, the avatar automatically joins the LiveKit room, begins streaming video, and stays synchronized with your agent's TTS output — no additional orchestration needed.
        <Tip>The avatar session lifecycle is tied to the room. Disconnect the room when your agent session ends to ensure a clean shutdown.</Tip>
      </Step>
    </Steps>

  </Tab>
</Tabs>
